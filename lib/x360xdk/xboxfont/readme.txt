Third Party Use of the MS Points Logo

The Microsoft Points name, along with the logo and symbol, are registered 
trademarks. To use the Microsoft Points logo, third parties must secure 
permission and a license agreement from Microsoft.

If your game uses or refers to Microsoft Points, you must follow TCR 133, 
and you must also follow the additional rules related to Microsoft Points 
naming and logo usages. Please contact your account manager since additional
licensing agreements apply.  See the following web page for more information, 

https://xds.xbox.com/xbox360/nav.aspx?page=xdksoftware/using_microsoft_points.htm
